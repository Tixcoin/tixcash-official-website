# Tixcash Official Website

Source code for the official Tixcash project website.

## Overview

The official website serves as the primary public-facing resource for the Tixcash ecosystem, providing:

- Project overview and vision
- Token distribution and economics documentation
- Roadmap and development milestones
- Team and foundation information
- Links to ecosystem tools (explorer, wallet, staking)

## Development

```bash
npm install
npm run dev
```
